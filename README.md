# Ticket Resolver

## Setup

1. Open the project folder.
2. Open Terminal.
3. Run:

```bash
python main.py
```

## Project Structure

- rag_retriever.py
- triage_agent.py
- knowledge_agent.py
- resolution_agent.py
- main.py

## Agents

1. Triage Agent
2. Knowledge Agent
3. Resolution Agent
4. Review Agent

The system processes support tickets and suggests solutions.
