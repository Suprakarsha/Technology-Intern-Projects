## 🔹 1. Simple LangChain

**File:** `simple_chain.py`

### Description

This project introduces the fundamentals of LangChain by connecting a Large Language Model with a Python application. User prompts are sent to the model, and AI-generated responses are displayed.

### Concepts Learned

* LangChain basics
* LLM invocation
* Prompt-response workflow

## 🔹 5. Retrieval-Augmented Generation (RAG) Chatbot

**File:** `rag_chatbot.py`

### Description

Developed a chatbot capable of answering questions from PDF documents by retrieving relevant information before generating responses.

### Workflow

User Question

↓

Retriever (FAISS)

↓

Relevant Document Chunks

↓

Google Gemini

↓

Generated Answer

### Concepts Learned

* Retrieval-Augmented Generation (RAG)
* Vector Search
* FAISS
* Embeddings
* PDF Processing
